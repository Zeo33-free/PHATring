# PHATRing

A JUCE-based audio plugin that sounds like Doppler distortion, but is actually a ring modulator + a unique clip algorithm.  
It can produce dry, crisp, brutal, and even fat sounds.

The soft clipping is not standard waveshaping — it’s designed for character rather than accuracy.
